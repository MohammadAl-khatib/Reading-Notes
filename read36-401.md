# DSA Review
## Whiteboard Advice

The most important advice (according to [article](https://hackernoon.com/the-best-whiteboard-interview-advice-i-ever-received-3ebbfa72e4a)) is to communicate, and here is how:

1. Restate the Question
2. Ask About Edge Cases
3. Ask About Test Cases
4. Write Pseudocode and Ask If It Makes Sense
5. Write the Actual Code and Ask if it Looks Good
6. Stuck? Ask for Help!

## 6 Tips
1. Take a few minutes.
2. Write down the steps of the solution.
3. Write pseudocode first.
4. Sit down. Be humble.
5. Come prepared.
6. Review your work.
